# ImageTools
      图片批量处理工具，包括按大小分类，按比例和尺寸剪裁，比较两个图片文件夹等

后续会完善说明，先列出主要功能：


 + filter: 按照图片大小过滤图片.
 + todir: 把图片按照指定数量移动到子目录中.
 + sync: 把一个目录的图片与另一个目录对比, 删除或移动第二个目录里比第一个目录中多出来的图片.
 + togallery: 把一个目录的图片可以按指定大小剪裁, 生成缩略图, 并可被Piwigo图库读取. 如果文件名中有tag，还会把tag放入图片IPTC元数据的标签中。
 
以上都是我自己处理图片的时候用到的功能。
